Política de Privacidade

Última atualização: 2026-02-23

A TrackerMap valoriza sua privacidade. Esta página descreve quais dados coletamos e por que os usamos.

Dados que coletamos

  • Dados técnicos (IP, user-agent) para segurança e operação do site.
  • Cookies/analytics (com consentimento) para medir tráfego e melhorar o serviço.
  • Dados fornecidos diretamente por você (nome, telefone, mensagem) quando preencher formulários.

Uso dos dados

Usamos os dados para operar o site, responder contatos, fornecer serviços e analisar uso para melhorias.

Consentimento

Ao aceitar o uso de cookies, autorizo a coleta e processamento de dados para finalidades de analytics. Você pode revogar o consentimento limpando o armazenamento local do navegador ou usando as opções no site.

Compartilhamento

Podemos compartilhar dados com provedores de serviços (ex.: Google Analytics) apenas para fins de analytics. Não vendemos dados pessoais.

Contato

Para dúvidas sobre privacidade, entre em contato: privacy@trackermap.example
